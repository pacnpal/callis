SSH Connection

Add this to your ~/.ssh/config to use Callis as a jump host. On Windows, this file is at C:\Users\<you>\.ssh\config.

Host callis
    HostName {{ ssh_host }}
    Port {{ ssh_port }}
    User {{ user.username }}
    IdentityFile ~/.ssh/your_ed25519_key

Replace ~/.ssh/your_ed25519_key with the path to your actual private key. Then connect to internal hosts with: ssh -J callis user@internal-host
