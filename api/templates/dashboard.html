SSH Connection

Add this to your ~/.ssh/config to use Callis as a jump host:

Host callis
    HostName {{ ssh_host }}
    Port {{ ssh_port }}
    User {{ user.username }}
    IdentityFile ~/.ssh/your_ed25519_key

Then connect to internal hosts with: ssh -J callis user@internal-host
